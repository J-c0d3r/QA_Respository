1. Quantas suítes de testes você desenvolveu?
Uma suíte contendo 8 casos de testes.

2. Os testes desenvolvidos são manuais ou automatizados?
São automáticos, bastando executar o comando 'mvn test -Dtest=Rick_MortyRunner'

3. Onde os testes se localizam na pirâmide apresentada?
Os testes se localizam na parte de Serviço porque foi testado o serviço(API's) de trazer as informações do Rick and Morty.

4. Os testes desenvolvidos são funcionais ou não-funcionais?
Os testes são funcionais.

5. Alguns dos testes desenvolvidos são testes Fim-a-Fim(End-To-End)?
Sim.

6. O que se deve fazer para que os testes desenvolvidos funcionem em modo regressão?
Para funcionar em modo regressão necessita executar todas as suítes de testes que foram criadas e testadas anteriormente 
com objetivo de encontrar alguma falha entre a última vez que os testes foram realizados e novamente executados.
